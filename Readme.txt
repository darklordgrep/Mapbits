Requirements - Oracle Application Express (APEX) version 22.2 or later.

Installation - Install the demo application, mapbits_demo_apex_application.sql, in your APEX web application. After install, you can add any of the plugins to your own application from "Shared Components" -> "Plugins". Click the "Create" button and select the option, "As a Copy of an Existing Plug-in".
Alternatively, you can install individual plugins directly from the APEX_Map_Region_Accessory_Plugins directory.

Documentation - For more information, documentation, and usace guides, refer to https://github.com/darklordgrep/Mapbits.
